#Gamu-Dama QA Portfolio 

This repository showcases my manual QA Testinng on a video game designed for USC's CTIN-489: Intermediate Game Design and Production, 
collaborating with Marcus Gordon. 

## Game Overview: 
- Platform: PC (Windows 10/11 & macOS)
- Mode: Single-Player
- Genre: Arcade, Beat-em-up
- Controls: 
    - Side Movement: A / D keys 
    - Aim & Shoot: Mouse 

## Skills Demostrated: 
- Manual Testing 
- Bug Reporting 
- Explorartory Testing 
- Gameplay Testing 
- Regression Testing 

## Test Areas Covered 
- Core Gameplay Mechanics 
- Save/Load Systems 
- System Performance
