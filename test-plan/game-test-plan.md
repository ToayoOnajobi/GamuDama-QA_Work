# Game Test Plan - Beat-em-up Arcade Game 

## 1. Objective 
Game delivers a stable, operational, and enjoyable experience for the players acrossed the supported platform, validated by gameplay mechanics, controls, NPC dialogue, and failure conditions. 

## 2. Game Overview
- Mode: Single-player 
- Engine: Unity
- Platform: PC (Windows & macOS)
- Player Mechanic: 
- Fail Condition: If Perwing's(NPC) health reaches 0. 

## 3. Test Scope
- Player's movement (A/D keys)
- Player's aim and shoot (Mouse)
- NPC health system 
- Win/Loss conditions 
- Enemy spawning and behavior
- UI elements (score summary, health bars, NPC dialogue)
- Performance and stability 
- Save/Load behavior


## 4. Test Types 
- Manual Testing 
- Exploratory Testing 
- Compatibility Testing (Windows/macOS)
- Performance Testing 
- Regression Testing 

## 5. Test Environment 
- Platform(s):
    - Windows 10/11 
    - macOS Ventura/Sonoma 
- Input Devices: 
    - Keyboard and Mouse
